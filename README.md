# Joel Maurice Lomnick Portfolio

Fresh personal portfolio website for Joel Maurice Lomnick, EIT, built with Next.js, TypeScript, the App Router, and Tailwind CSS.

## Run Locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Production Build

```bash
pnpm build
pnpm start
```
